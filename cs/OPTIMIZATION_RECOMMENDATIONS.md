## 🎯 具体优化方案

### 方案1: 性能优化 (Speed Up)

#### 1.1 移除 Float Diffusion
```
当前: Float Diffusion + U32_ARX + S-Box
优化: 移除 Float Diffusion，仅保留 U32_ARX + S-Box

预期效果:
- 加密时间: 1.53s → 1.0s (⬇️ 35%)
- 安全性: NPCR 99.6% 保持不变
- 重构质量: 不变
```

#### 1.2 使用静态 S-Box
```
当前: 动态 S-Box (每块生成)
优化: 预计算静态 S-Box

预期效果:
- 解密时间: 11s → 8s (⬇️ 27%)
- 安全性: 几乎不变
- 内存: 减少
```

#### 1.3 优化 FISTA 参数
```python
# 当前参数
lam = 0.005
iter = 200

# 优化建议
lam = 0.003  # 降低正则化，提升细节
iter = 150    # 减少迭代，速度提升
```

### 方案2: 质量优化 (Quality Up)

#### 2.1 提高压缩率
```
当前: cr = 0.5 (50% 压缩)
优化: cr = 0.7 (30% 压缩)

实验结果:
- cr=0.5: PSNR=30.71dB, SSIM=0.883
- cr=0.7: PSNR=33.71dB, SSIM=0.932 ✅

提升: SSIM +5.5%, PSNR +3dB
```

#### 2.2 自适应 Lambda
```python
# 根据图像内容自适应调整
def adaptive_lambda(image):
    variance = np.var(image)
    if variance > 5000:
        return 0.003  # 细节丰富的图像
    else:
        return 0.01    # 平滑图像
```
