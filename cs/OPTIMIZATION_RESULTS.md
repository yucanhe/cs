## 🎯 推荐配置

### 方案1: 平衡型 (推荐)
```bash
python image/demo/demo3.py \
    --img resources/images/lena.png \
    --key "my-secret-key-2026" \
    --cr 0.7 \
    --lam 0.005 \
    --iter 200 \
    --ablate no_float_diff \
    --out results/optimized/balanced
```

**效果:**
- PSNR: 33.71 dB (+9.8%)
- SSIM: 0.932 (+5.5%)
- NPCR: 99.63% (保持)
- 加密时间: 2.2s (+44%)

### 方案2: 高质量型
```bash
python image/demo/demo3.py \
    --img resources/images/lena.png \
    --key "my-secret-key-2026" \
    --cr 0.9 \
    --lam 0.005 \
    --iter 200 \
    --ablate no_float_diff \
    --out results/optimized/high_quality
```

**效果:**
- PSNR: 36.10 dB (+17.5%)
- SSIM: 0.956 (+8.3%)
- NPCR: 99.62% (保持)
- 加密时间: 2.7s (+76%)

## ✅ 优化结论

1. **提高 CR 是提升质量最有效的方法**
   - CR 0.5→0.9: SSIM 0.883→0.956 (+8.3%)
   - CR 0.5→0.9: PSNR 30.71→36.10 dB (+17.5%)

2. **安全性保持不变**
   - 所有 CR 下 NPCR > 99.6%
   - UACI > 33.4%

3. **权衡**
   - 更高 CR = 更好质量 = 更长加密时间
   - 推荐 CR=0.7 作为平衡点

4. **迭代次数 ITER 影响较小**
   - iter=150 vs 200 差异 < 0.5dB
   - 建议保持 200 以确保收敛
