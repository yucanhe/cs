### 🎯 推荐配置

#### 高质量配置 (推荐)
```bash
python audio/audio_cs_spcmm_crypto.py encrypt \
    --in_wav resources/audio/1.wav \
    --out_npz results/optimize_audio/audio_high.npz \
    --key "my-secret-key-2026" \
    --m_rate 0.95 \
    --iter 300
```

- **PSNR**: 22.39 dB (+1.4%)

#### 高速配置

## 🎬 视频优化参数

### 可调参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| **m_rate_key** | 0.7 | 0.5-1.0 | 关键帧压缩率 |
| **key_stride** | 2 | 1-5 | 关键帧间隔 |
| **iters** | 100 | 50-200 | FISTA迭代 |

### 预期效果

```
m_rate_key 影响:
- 0.5: 高压缩, 质量略低
- 0.7: 平衡 (当前)
- 0.9: 低压缩, 质量高

key_stride 影响:
- 1: 每帧是关键帧, 质量最高
- 2: 每2帧关键帧 (当前)
- 3-5: 压缩率更高, 质量降低
```
